## Description

The experiment illustrates how spectral overlap between molecular templates can generate structural degeneracy in atmospheric retrieval architectures.

A synthetic transmission spectrum is generated using H₂O, CH₄ and a simple cloud component.  
Alternative molecular bases with tunable spectral similarity are introduced in order to map how statistical distinguishability between retrieval architectures varies as a function of spectral overlap.
